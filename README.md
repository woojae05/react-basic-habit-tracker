# habit-traker

> 이 프로젝트는 드림코딩 강의를 보고 제작하였습니다.

## 기능

1. 자신의 습관을 등록, 삭제 할 수 있다.
2. 습관의 카운팅을 할 수 있다.
3. 습관의 전체 수를 확인 할 수 있다.

---

## 배운점

react라이브러리 사용해 간단한 프로젝트를 만들고 배포하는 과정을 차근차근 배웠습니다.
